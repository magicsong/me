## 🌟 项目缘起
在深度实践《认知觉醒》的过程中，我发现：
- 90%的成长困境源于「道理都懂但做不到」
- 「元认知」需要可视化工具才能持续激活
- 本能脑/情绪脑/理智脑的博弈需要数字结界

于是将书中7大核心概念（元认知、拉伸区、心智带宽...）编译成可执行的数字工具，打造认知进化的「行为编译器」
